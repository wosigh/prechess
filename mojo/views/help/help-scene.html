This version of chess is 100% javascript by Neil Pearce.

The original source can be found at http://javascript.internet.com/games/javascript-chess-with-cpu-oppo.html.

Castling, en-passent and pawn promotion are all catered for, but the rule "Game is drawn if a position repeats three times" is not implemented. The CPU performs an alpha-beta search to a depth of two moves, but does not examine check at the deepest level.

Tap a piece to indicate that it should move. Tap a cell to drop it.

Game log appears below the board. Scroll to read it.

This game's version of end-game strategy is to win before the end-game.

I hope you enjoy it. If you can improve the game engine, go for it!
